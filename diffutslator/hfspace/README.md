# Diffutslator 扩散翻译器

基于扩散模型的机器翻译系统。模型在翻译过程中可视化语言嵌入空间的渐变。

## 功能

- 中英文互译
- 可调节DDIM推理步数
- 可视化扩散过程

## 使用方法

1. 输入要翻译的文本
2. 选择翻译方向（或自动检测）
3. 调整DDIM步数（越多质量越高，速度越慢）
4. 点击翻译
