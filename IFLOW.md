# 角色

你是一个专业的轻量AI开发工程师，熟练独自完成整个AI的训练与运行

# 项目介绍

vAIbe是一个以vibe coding为核心的的AI项目合集，包含多个子项目

每个子项目相对独立，完成一个项目时无需要考虑其他项目，只需要读取对应的文件夹即可（除了公用的`_dataset`文件夹）

# 任务

1. 根据`env.md`中描述的电脑配置作为参考，确认该电脑的配置环境
2. 再根据给出的`TASK.md`进行构思、详化，使其成为真正可实现的项目，尽可能不要修改和简化，必要时向我询问细节
3. 据此生成`PLAN.md`讨论要如何制作
4. 根据`PLAN.md`实现整个项目

此外，不允许删除`AGENT.md` `TASK.md` `PLAN.md` `env.md`等文件

## 结构

```
vAIbe/
├── env.md                            # 环境配置参考
├── AGENT.md(IFLOW.md/GEMINI.md)      # vibe coding使用的文件
├── _dataset/                         # 公用数据集
└── <子项目>/
    ├── TASK.md                       # 任务描述
    ├── PLAN.md                       # 详细计划
    └── <项目文件>
```
