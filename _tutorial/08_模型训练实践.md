## 8.1 训练前的准备

### 明确目标

```
在开始之前，先回答这些问题：

1. 要解决什么问题？
   └── 分类？检测？生成？

2. 有什么数据？
   └── 多少样本？质量如何？有标注吗？

3. 评估标准是什么？
   └── 准确率？速度？还是两者兼顾？

4. 资源限制？
   └── 你的电脑：8GB内存，CPU训练
   └── 训练时间能接受多久？

5. 部署环境？
   └── 本地运行？服务器？手机APP？
```

### 可行性判断

**在你的电脑上可行的任务：**
```
✅ 传统机器学习（sklearn）
   └── 数据量：<10万样本
   └── 特征维度：<1000

✅ 小型神经网络训练
   └── 参数量：<1000万
   └── 图像尺寸：<224×224
   └── 数据量：<1万张

✅ 预训练模型微调
   └── 冻结大部分层
   └── 只训练最后一层或分类头

✅ 大模型量化推理
   └── 使用llama.cpp/Ollama
   └── 不做训练，只做推理
```

**在你的电脑上不可行的任务：**
```
❌ 大模型预训练（>1B参数）
   └── 需要：多卡A100，数月时间

❌ 大模型全参数微调
   └── 需要：至少24GB显存

❌ 大批量训练
   └── 8GB内存限制

❌ 高分辨率图像训练（>512×512）
```

## 8.3 模型训练流程

### 标准训练代码结构

```python
import torch
import torch.nn as nn
from torch.utils.data import DataLoader
from tqdm import tqdm

# 1. 定义模型
class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        # 定义层
        self.fc1 = nn.Linear(784, 256)
        self.fc2 = nn.Linear(256, 10)
        self.relu = nn.ReLU()
        self.dropout = nn.Dropout(0.2)
    
    def forward(self, x):
        x = self.relu(self.fc1(x))
        x = self.dropout(x)
        x = self.fc2(x)
        return x

# 2. 准备数据
train_loader = DataLoader(train_dataset, batch_size=32, shuffle=True)
val_loader = DataLoader(val_dataset, batch_size=32)

# 3. 初始化
model = MyModel()
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
scheduler = torch.optim.lr_scheduler.StepLR(optimizer, step_size=10, gamma=0.1)

# 4. 训练循环
num_epochs = 50
best_val_acc = 0

for epoch in range(num_epochs):
    # 训练阶段
    model.train()
    train_loss = 0
    train_correct = 0
    train_total = 0
    
    for batch_idx, (data, target) in enumerate(tqdm(train_loader)):
        # 前向传播
        output = model(data)
        loss = criterion(output, target)
        
        # 反向传播
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        # 统计
        train_loss += loss.item()
        _, predicted = output.max(1)
        train_total += target.size(0)
        train_correct += predicted.eq(target).sum().item()
    
    # 验证阶段
    model.eval()
    val_loss = 0
    val_correct = 0
    val_total = 0
    
    with torch.no_grad():
        for data, target in val_loader:
            output = model(data)
            loss = criterion(output, target)
            
            val_loss += loss.item()
            _, predicted = output.max(1)
            val_total += target.size(0)
            val_correct += predicted.eq(target).sum().item()
    
    # 计算指标
    train_acc = 100. * train_correct / train_total
    val_acc = 100. * val_correct / val_total
    
    print(f'Epoch {epoch+1}/{num_epochs}:')
    print(f'  Train Loss: {train_loss/len(train_loader):.4f}, Acc: {train_acc:.2f}%')
    print(f'  Val Loss: {val_loss/len(val_loader):.4f}, Acc: {val_acc:.2f}%')
    
    # 学习率调整
    scheduler.step()
    
    # 保存最佳模型
    if val_acc > best_val_acc:
        best_val_acc = val_acc
        torch.save(model.state_dict(), 'best_model.pth')
        print(f'  ✓ Saved best model (val_acc: {val_acc:.2f}%)')
```

### 关键技巧

**1. 早停（Early Stopping）**
```python
# 当验证集性能不提升时停止训练
patience = 5  # 容忍5轮不提升
no_improve = 0

if val_acc > best_val_acc:
    best_val_acc = val_acc
    torch.save(model.state_dict(), 'best_model.pth')
    no_improve = 0
else:
    no_improve += 1
    if no_improve >= patience:
        print('Early stopping triggered')
        break
```

**2. 检查点保存**
```python
# 定期保存，防止训练中断丢失进度
checkpoint = {
    'epoch': epoch,
    'model_state_dict': model.state_dict(),
    'optimizer_state_dict': optimizer.state_dict(),
    'best_acc': best_val_acc,
}
torch.save(checkpoint, f'checkpoint_epoch_{epoch}.pth')
```

**3. 混合精度训练（节省显存）**
```python
from torch.cuda.amp import autocast, GradScaler

scaler = GradScaler()

for data, target in train_loader:
    optimizer.zero_grad()
    
    with autocast():  # 自动混合精度
        output = model(data)
        loss = criterion(output, target)
    
    scaler.scale(loss).backward()
    scaler.step(optimizer)
    scaler.update()
```

## 8.4 超参数调优

### 关键超参数

| 超参数 | 作用 | 经验值 | 调整建议 |
|--------|------|--------|----------|
| **Learning Rate** | 学习速度 | 1e-4 ~ 1e-2 | 太大不收敛，太小太慢 |
| **Batch Size** | 每次迭代样本数 | 16-256 | 越大越稳定，需配合调整LR |
| **Epochs** | 训练轮数 | 视任务而定 | 配合早停使用 |
| **Optimizer** | 优化算法 | Adam/AdamW | 默认AdamW |
| **Dropout** | 正则化强度 | 0.2-0.5 | 过拟合严重时增大 |
| **Weight Decay** | L2正则 | 1e-5 ~ 1e-3 | 防止过拟合 |

### 学习率调优

**学习率范围测试：**
```python

# 画出lr vs loss曲线，选择损失下降最快的lr范围
```

**学习率策略：**
```python

# 2. Cosine Annealing

# 3. Reduce on Plateau（推荐）

    optimizer, mode='min', factor=0.5, patience=5

# 4. Warmup + Cosine

    optimizer, num_warmup_steps=100, num_training_steps=1000
)
```

## 8.5 调试训练问题

### 问题1：损失不下降

```
可能原因:
├── 学习率太小 → 调大学习率
├── 学习率太大 → 调小学习率
├── 梯度消失 → 检查激活函数、归一化
├── 数据问题 → 检查标签、归一化
└── 代码bug → 检查损失计算、反向传播

检查清单:
✓ 打印初始损失，看是否合理
✓ 检查梯度是否存在（是否None）
✓ 检查参数是否更新
✓ 用一个样本过拟合测试
```

### 问题2：过拟合

```
表现:
训练集准确率 ↑↑↑
验证集准确率 → 或 ↓

解决方案（按优先级）:
1. 增加数据
2. 数据增强
3. 正则化（Dropout、L2）
4. 早停
5. 简化模型
6. 集成学习
```

### 问题3：训练不稳定

```
表现:
损失震荡很大
验证指标忽高忽低

解决方案:
├── 减小学习率
├── 增大batch size
├── 使用学习率warmup
├── 使用梯度裁剪
└── 检查数据shuffle
```

## 8.6 使用预训练模型

### 为什么要用预训练模型？

```
优势:
├── 节省时间（不用从头训练）
├── 效果更好（大数据预训练）
├── 数据需求少（小样本也能好）
└── 泛化能力强

适用场景:
├── 你的数据少（<1万张图）
├── 计算资源有限
├── 想快速验证想法
└── 大多数实际应用
```

### 微调策略

**策略1：特征提取**
```python
# 冻结预训练模型，只训练分类头
model = torchvision.models.resnet50(pretrained=True)

# 冻结所有层
for param in model.parameters():
    param.requires_grad = False

# 替换最后的分类层
model.fc = nn.Linear(model.fc.in_features, num_classes)

# 只训练最后一层
optimizer = torch.optim.Adam(model.fc.parameters(), lr=0.001)
```

**策略2：微调全部**
```python
# 加载预训练权重
model = torchvision.models.resnet50(pretrained=True)
model.fc = nn.Linear(model.fc.in_features, num_classes)

# 使用较小的学习率
optimizer = torch.optim.Adam([
    {'params': model.layer1.parameters(), 'lr': 1e-5},
    {'params': model.layer2.parameters(), 'lr': 1e-5},
    {'params': model.layer3.parameters(), 'lr': 1e-4},
    {'params': model.layer4.parameters(), 'lr': 1e-4},
    {'params': model.fc.parameters(), 'lr': 1e-3}
])
```

**策略3：逐层解冻**
```python
# 先训练分类头
# 然后逐层解冻预训练层
for param in model.layer4.parameters():
    param.requires_grad = True
# 训练几轮...

for param in model.layer3.parameters():
    param.requires_grad = True
# 训练几轮...
```
